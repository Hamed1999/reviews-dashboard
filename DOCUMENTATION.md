# FlexLiving Reviews Dashboard Documentation

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- React Icons

## Key Design Decisions

1. **Approval System**: Used localStorage for simplicity
2. **UI/UX**: Mimicked FlexLiving's clean, modern aesthetic
3. **Data Normalization**: Transformed Hostaway's nested structure to flat review objects

## API Behaviors

### `/api/reviews/hostaway`

- Fetches and normalizes mock Hostaway data
- Handles missing fields with defaults
- Returns structured JSON
